# Linux Fundamentals

## What is Linux?

Linux is an open-source operating system widely used in servers, cloud computing, DevOps, and container technologies.

## Why Linux is Important for DevOps?

- Most cloud servers run Linux
- Docker containers are Linux-based
- Kubernetes clusters use Linux nodes
- CI/CD pipelines often run on Linux
- Automation and scripting rely on Linux commands
