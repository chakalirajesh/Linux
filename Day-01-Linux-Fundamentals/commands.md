# Day 1 Commands

## pwd
Print current working directory.

Example:
pwd

## ls
List files and directories.

Examples:
ls
ls -l
ls -la

## cd
Change directory.

Examples:
cd Documents
cd ..
cd ~

## mkdir
Create directories.

Example:
mkdir project

## touch
Create files.

Example:
touch file.txt

## cat
Display file contents.

Example:
cat file.txt

## cp
Copy files.

Example:
cp file1.txt file2.txt

## mv
Move or rename files.

Example:
mv old.txt new.txt

## rm
Delete files.

Example:
rm file.txt

## grep
Search text patterns.

Example:
grep "hello" file.txt

## find
Search files and folders.

Example:
find . -name "*.txt"
