# File and Directory Management

## Overview

File and directory management is one of the most important Linux skills. These commands help users create, organize, move, copy, and delete files efficiently.

## Creating Directories

Create a new directory:

```bash
mkdir project
```

Create multiple directories:

```bash
mkdir dir1 dir2 dir3
```

## Creating Files

Create an empty file:

```bash
touch file.txt
```

## Copying Files

Copy a file:

```bash
cp source.txt destination.txt
```

Copy a directory:

```bash
cp -r folder1 folder2
```

## Moving Files

Move a file:

```bash
mv file.txt Documents/
```

Rename a file:

```bash
mv old.txt new.txt
```

## Removing Files

Delete a file:

```bash
rm file.txt
```

Delete a directory:

```bash
rm -r folder
```

## Viewing Directory Structure

```bash
tree
```

## Conclusion

Linux file management commands are essential for DevOps engineers because they are used daily while managing servers, applications, and automation scripts.
