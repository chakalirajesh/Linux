# Practice Commands

```bash
mkdir Day2-Practice

cd Day2-Practice

touch file1.txt file2.txt

ls -la

cp file1.txt copy_file1.txt

mkdir Backup

mv copy_file1.txt Backup/

mv file2.txt renamed_file2.txt

tree

rm renamed_file2.txt

tree
```

## Commands Used

| Command | Purpose                  |
| ------- | ------------------------ |
| mkdir   | Create directory         |
| touch   | Create file              |
| ls      | List files               |
| cp      | Copy file                |
| mv      | Move/Rename file         |
| rm      | Delete file              |
| tree    | Show directory structure |

## Learning Outcome

* Created directories and files
* Copied files
* Moved files between directories
* Renamed files
* Deleted files
* Visualized directory structures
