# System Information Commands

## Overview

Linux provides commands to monitor CPU, memory, storage, and operating system information.

## Check Operating System

```bash
uname -a
```

## Hostname

```bash
hostname
```

## Memory Usage

```bash
free -h
```

## Disk Usage

```bash
df -h
```

## Directory Size

```bash
du -sh .
```

## CPU Information

```bash
lscpu
```

## Kernel Version

```bash
uname -r
```

## Uptime

```bash
uptime
```

## View Mounted File Systems

```bash
mount
```

## Learning Outcome

* Gather system information
* Monitor memory and storage
* Check OS and kernel details
* Understand server resources
* Prepare for troubleshooting tasks
